## 下载

### Docker

```
docker pull apachecn0/apachecn-devops-zh
docker run -tid -p <port>:80 apachecn0/apachecn-devops-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g apachecn-devops-zh
apachecn-devops-zh <port>
# 访问 http://localhost:{port} 查看文档
```
